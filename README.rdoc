== MiamiYum

Aggregated daily deals for only restaurants in Miami.

== Features + Functionality
- Uses the Sqoot API to get deals
- Incorporated Bootstrap for clean and responsive design
- Uses gmaps4rails gem to generate map
- Incorporates Mailchimp to collect email address
